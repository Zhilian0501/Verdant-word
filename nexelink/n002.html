影片剪輯任務

請根據提供素材製作 1 分鐘短影片，需輸出 mp4 與簡短說明。

前端開發任務

請製作首頁與登入頁 RWD 響應式，支援桌機與手機瀏覽。

全端開發任務

負責一套簡易會員系統，包括前端與後端串接、資料庫連接。

後端開發任務

使用 Node.js 架設 API 並串接資料庫，驗證登入與儲存功能。

平面設計任務

請設計商品主視覺一張，JPG/PNG 格式，需附上構圖說明。

APP 開發任務

請設計 Flutter 或 React Native APP 原型，支援登入與地圖。
